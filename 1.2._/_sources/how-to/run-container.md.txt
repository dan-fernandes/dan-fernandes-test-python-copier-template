# Run in a container

Pre-built containers with dan-fernandes-test-python-copier-template and its dependencies already
installed are available on [Github Container Registry](https://ghcr.io/dan-fernandes/dan-fernandes-test-python-copier-template).

## Starting the container

To pull the container from github container registry and run:

```
$ docker run ghcr.io/dan-fernandes/dan-fernandes-test-python-copier-template:latest --version
```

To get a released version, use a numbered release instead of `latest`.
